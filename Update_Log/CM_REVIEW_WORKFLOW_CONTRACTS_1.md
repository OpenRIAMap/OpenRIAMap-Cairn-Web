# CM_REVIEW_WORKFLOW_CONTRACTS_1

This contract-only change is based on the approved `CM_EMBEDDED_REVIEW_WORKBENCH_FRAME_1` behavior baseline. It does not modify Review UI, MapContainer, Mapping behavior, panel layout, labels, button placement, or user interactions.

It defines generic seam types for a future application-owned Review workflow: host arbitration, temporary-layer mounting, inbox/intents, session state, local adapter registration, configuration extension shape, and a frozen upstream showcase-template boundary.

No production approval, external data connection, cloud operation, or downstream configuration synchronization is implemented. A later frontend change requires a separate user-specified UI request and must retain the baseline module style.
